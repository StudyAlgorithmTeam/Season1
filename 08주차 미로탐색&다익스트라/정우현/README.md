bfs를 사용하여 풀었으며 좌표 대신 도시의 번호와 거리를 변수로 사용하였다.

그러다 거리인 n 값이 K와 같아 지면 결과 배열인 a에 넣어준다. n이 K를 넘어가게 되면
더 이상 확인할 필요가 없으므로 종료한다.
배열이 비었다면 -1을 출력하고 아니면 정렬 후 출력해준다.
